IOM

概述

IOM Input Output Monitor ，是芯片IO口监控模块，用于监控外设输入输出信号是否满足期望时序。如果不满足就报错。

功能框图

image-20230406115021845

特性

FPC滤波预分频器

LAM逻辑分析

ECM事件组合模块

  • 16 个过滤预分频通道(Filter&Prescaler Channels),每一个都连接到一个特定的 pad(Pn _IN)，通道的输出可以作为一个监视或参考信号。
  • 16个逻辑分析模块(Logic Analyzer Modules)，每个通道包合一个mux，用于独立的选择监控和参考信号，并与之进行比较。当所选的监控和参考信号相同或不同的时候，就会产生一个本地事件。可以通过寄存器设置进行配置。
  • 一个EXOR 组合器，可配置(通过寄存器) 选择8个GTM 输入来产生一个组合信号作为一个参考信号。
  • 事件组合模块 (Event Combiner Module)，把16 个本地事件信号(来自16个LAM)合成一个系统事件信号 (连接着系统安全管理模块)，可通过寄存器配置。
  • 系统外设总线接口 (System Peripheral Bus)，用于配置内部的控制状态寄存器。
